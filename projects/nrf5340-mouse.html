Custom NRF5340 Wireless Gaming Mouse and High-Speed USB Dongle

A complete custom mouse and receiver system designed from hardware to software, supporting wired USB communication and low-latency wireless communication through Nordic Enhanced ShockBurst, with a 4 kHz wireless refresh-rate target and a dual-MCU high-speed USB dongle.

Project Overview

This project is a custom high-performance wireless mouse ecosystem consisting of a custom NRF5340-based mouse, a custom NRF5340 plus NXP LPC55S28 receiver dongle, and PC-side configuration software. The system was designed to support both wired USB operation and wireless operation through Nordic Enhanced ShockBurst communication, targeting a 4 kHz refresh rate for low-latency input performance.

Custom Mouse Hardware

The mouse PCB was designed around the NRF5340 SoC, PAW3395 optical sensor, and NPM1300 power management IC. The design includes motion sensing, wireless communication, USB connectivity, battery charging, battery monitoring, and embedded firmware support.

High-Speed Receiver Dongle

The dongle combines an NRF5340 wireless radio interface with an NXP LPC55S28 microcontroller for high-speed USB HID communication. This architecture was selected to overcome the NRF5340 full-speed USB limitation while preserving NRF wireless capability.

PC Configuration Software

A dedicated PC-side host application was developed to configure mouse settings and interact with the device from the computer. The software supports practical configuration without requiring the internal source code to be released publicly.

System Architecture

The system uses a split architecture to balance wireless performance, USB capability, and power efficiency. The mouse collects motion data and sends input reports through USB or wireless communication. In wireless mode, the dongle receives radio packets through the NRF5340 and passes processed data to the LPC55S28 through high-speed SPI for high-speed USB HID reporting to the PC.

Why the Dual-MCU Dongle Was Used

The NRF5340 provides the wireless communication capabilities required for the mouse receiver, but its USB interface is limited to full-speed USB. To support high-speed HID communication, the dongle integrates the LPC55S28 as a dedicated high-speed USB interface controller. The NRF5340 receives wireless input packets and forwards processed data to the LPC55S28 through a high-speed SPI connection using a custom messaging protocol. This allows the system to combine Nordic wireless communication with high-speed USB HID reporting in a single compact receiver.

Professional Project Description

This project involved the complete design and development of a custom high-performance wireless mouse system consisting of an NRF5340-based mouse, an NRF5340 and NXP LPC55S28-based receiver dongle, and PC-side configuration software. The mouse supports both direct USB communication and wireless communication using Nordic Enhanced ShockBurst, with the wireless system designed for a 4 kHz refresh rate. Since the NRF5340 only supports full-speed USB, the receiver dongle was designed with a dual-microcontroller architecture: the NRF5340 handles wireless communication, while the LPC55S28 provides high-speed USB HID communication with the host computer. The two microcontrollers communicate through a high-speed SPI interface using a custom messaging protocol.

The mouse hardware integrates a PAW3395 optical sensor for motion sensing and an NPM1300 power management IC for battery charging, monitoring, and power management. Both the mouse and dongle were designed from the ground up, including hardware architecture, schematic design, PCB layout, firmware architecture, communication protocol design, and system integration. A PC-side host application was also developed to allow configuration of mouse settings. Although the source code is not publicly released, the project demonstrates end-to-end embedded systems development across wireless communication, USB HID design, sensor integration, battery-powered hardware, inter-processor communication, and desktop software tooling.
